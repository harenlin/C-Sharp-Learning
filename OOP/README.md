### How To Run?
Part 1. Inheritance
```
rm *.exe
csc Person.cs Customer.cs Program.cs
mono Program.exe
```

Part 2.
```
rm *.exe
csc Http*
mono HttpProgram.exe
```

Part 3. Up/Down-Casting
```
rm *.exe
csc Car.cs Vehicle.cs CarProgram.cs 
mono CarProgram.exe
```

Part 4. virtual / abstract / override
```
rm *.exe
csc ShapeProgram.cs
mono ShapeProgram.exe
```
